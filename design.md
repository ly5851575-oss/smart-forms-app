# تصميم تطبيق نماذج ذكية

## قائمة الشاشات

| الشاشة | الوصف | الوظائف |
|--------|-------|--------|
| **LoginScreen** | شاشة الاختيار بين طرق الدخول | اختيار تسجيل الدخول بجوجل أو البريد |
| **EmailLoginScreen** | تسجيل الدخول بالبريد | إدخال البريد وكلمة المرور |
| **RegisterScreen** | إنشاء حساب جديد | إدخال البيانات والتحقق |
| **EmailVerificationScreen** | التحقق من البريد | عرض رسالة التحقق وزر التحقق |
| **ForgotPasswordScreen** | استعادة كلمة المرور | إدخال البريد وإرسال رابط الاستعادة |
| **HomeScreen** | الصفحة الرئيسية | عرض بيانات المستخدم والخروج |

## المحتوى الأساسي والوظائف

### LoginScreen
- عنوان "نماذج ذكية"
- شعار التطبيق
- زر "تسجيل الدخول بجوجل"
- زر "تسجيل الدخول بالبريد الإلكتروني"
- رابط "إنشاء حساب جديد"

### EmailLoginScreen
- حقل البريد الإلكتروني
- حقل كلمة المرور
- زر "تسجيل الدخول"
- رابط "نسيت كلمة المرور؟"
- رابط "العودة"

### RegisterScreen
- حقل الاسم الكامل
- حقل البريد الإلكتروني
- حقل كلمة المرور
- حقل تأكيد كلمة المرور
- زر "إنشاء الحساب"
- رابط "لديك حساب بالفعل؟ سجل الدخول"

### EmailVerificationScreen
- رسالة "تم إرسال رسالة تحقق إلى بريدك الإلكتروني"
- عرض البريد المدخل
- زر "تحققت من البريد"
- زر "إعادة إرسال رسالة التحقق"
- رابط "العودة"

### ForgotPasswordScreen
- حقل البريد الإلكتروني
- زر "إرسال رابط الاستعادة"
- رسالة النجاح
- رابط "العودة"

### HomeScreen
- عرض اسم المستخدم
- عرض البريد الإلكتروني
- عرض حالة الاشتراك (تجربة مجانية / مشترك)
- عرض تاريخ انتهاء التجربة
- زر "تسجيل الخروج"

## تدفقات المستخدم الرئيسية

### تدفق تسجيل الدخول بجوجل
1. المستخدم يضغط "تسجيل الدخول بجوجل"
2. يفتح اختيار حساب Google
3. بعد النجاح، يتم حفظ البيانات في Firestore
4. الانتقال إلى HomeScreen

### تدفق إنشاء حساب بالبريد
1. المستخدم يضغط "إنشاء حساب جديد"
2. يملأ البيانات (الاسم، البريد، كلمة المرور)
3. التحقق من صحة البيانات
4. إنشاء الحساب في Firebase Auth
5. إرسال رسالة التحقق
6. الانتقال إلى EmailVerificationScreen
7. بعد التفعيل، الانتقال إلى HomeScreen

### تدفق تسجيل الدخول بالبريد
1. المستخدم يدخل البريد وكلمة المرور
2. التحقق من البيانات
3. إذا لم يتم تفعيل البريد، الانتقال إلى EmailVerificationScreen
4. إذا تم تفعيل البريد، الانتقال إلى HomeScreen

- **الحد الأقصى للعرض**: 480px
- **حجم الخط الأساسي**: 16px

- **حجم العنوان الفرعي**: 18px
